# 马尾食堂 - 移动端电商APP原型

基于 HTML + Tailwind CSS + FontAwesome + 原生 JavaScript 构建的中国移动员工食堂购物APP原型。

## 文件结构

```
canteen-app/
├── index.html              # 主入口（iframe 平铺展示所有页面）
├── 首页.html               # 首页
├── 搜索页.html             # 搜索页
├── 全品类分类页.html        # 全品类分类页
├── 商品详情页.html          # 商品详情页
├── 购物车.html             # 购物车
├── styles.css             # 共享样式
└── app.js                 # 共享 JavaScript（数据、工具函数）
```

## 快速预览

直接在浏览器中打开 `index.html` 即可预览所有页面。

也可以单独打开任意 HTML 文件独立预览。

## 技术栈

- **HTML5** - 语义化结构
- **CSS3** - Tailwind 风格工具类 + CSS 变量 + Flexbox/Grid 布局
- **FontAwesome 6.5** - 图标库
- **原生 JavaScript** - 无框架依赖，ES6+ 语法

## 功能特性

### 首页
- 顶部定位栏（马尾食堂）+ 搜索入口 + 消息中心
- Banner 轮播（自动播放 + 手动滑动）
- 公告滚动条
- 金刚区（自选区、自制品、助农产品、限时团购）
- 热门专区（卤味、面点、健康粥品、节日礼盒）
- 横向滚动推荐商品
- 精选好物网格（无限下滑加载）
- 底部 Tab 导航栏
- 购物车徽章实时更新

### 搜索页
- 热门搜索榜单（带排名标识）
- 搜索历史记录（支持单条删除 + 一键清空）
- 实时搜索联想词
- 搜索结果网格展示
- 综合/销量/价格/新品排序
- 侧边筛选面板（品类、价格区间、优惠）
- 无结果时的相似商品推荐

### 全品类分类页
- 左侧一级分类固定导航（面点/卤味/糕点/自制品）
- 顶部二级分类横向标签栏
- 综合/销量/价格排序 + 筛选入口
- 商品网格布局
- 商品卡片一键加购（Toast 反馈）
- 加购成功提示

### 商品详情页
- 商品大图展示
- 价格信息 + 折扣标签 + 销量信息
- 配送信息展示
- 规格选择入口
- 店铺信息卡片
- 商品详情描述
- 用户评价列表
- SKU 选择弹窗（规格 + 数量）
- 加入购物车 / 立即购买
- 悬浮底部操作栏

### 购物车
- 品类分区展示
- 商品勾选状态
- 商品数量增减
- 全选/反选
- 满减/免配送费门槛提示
- 无货商品自动置灰 + 单独分区
- 清空无货商品
- 实时金额核算
- 删除已选商品
- 去结算按钮

## 响应式设计

- 设计宽度：**375px**（标准移动端）
- 最大宽度：**430px**（大屏手机）
- 使用 CSS `max-width` + `env(safe-area-inset-bottom)` 确保刘海屏适配
- 所有布局使用 Flexbox + Grid，确保不溢出

## 主色调

- **#1890FF** - 品牌主色（蓝色）
- **#F5222D** - 价格/促销色
- **#52C41A** - 成功色
- **#FAAD14** - 警告色

## 数据说明

- 所有商品数据为 Mock 数据，存储在 `app.js` 的 `PRODUCTS` 数组中
- 购物车数据使用 `localStorage` 持久化
- 搜索历史使用 `localStorage` 持久化

## 交互规范

- 所有可点击元素有 active 态（缩放/透明度变化）
- 页面转场有顺滑动画
- 操作结果有 Toast 提示（2秒自动消失）
- 列表支持下拉刷新/上滑加载
- 空状态、加载状态、异常状态均有处理
